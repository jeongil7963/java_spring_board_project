# 스프링
	자바 프레임 워크를 이용하여 게시판을 구현하기 위한 기본 예제와 프로젝트를 진행하였다.
	
	가장 기본적인 스프링 컨테이너 및 설정 파일 구성법에 대하여 공부를 하였고
	
	의존성 주입과 어노테이션 기반 설정에 대해 공부를 하였다.
	
	그 개념을 바탕으로 실질적인 비즈니스 컴포넌트를 실습하여 Back-end 구성과 로직에 대하여 공부를 하였다.
	
	그 후 AOP 개념을 공부하여 스프링의 강력함에 대해 이해할 수 있었다.
	
	JDBC,트랜잭션 처리를 공부한 후에
	
	모델 1,2 아키텍처의 구조와 현재 MVC 프레임워크로 구현을 하면서 MVC 적용법에 대해 익힐 수 있었다.
	
	검색기능, 파일 업로드, 다국어 처리, 데이터 변환 등 많은 분야를 프로젝트를 진행하면서 이해할 수 있는 좋은 계기가 되었다.
